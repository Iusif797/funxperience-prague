# FunXperience Prague

FunXperience Prague is an ultra-premium, cinematic landing page designed for booking and customizing the perfect bachelor party experience in Prague.

## Features

- **Cinematic Parallax Hero**: A multi-layered interactive parallax background presenting the gorgeous night view of Prague, dynamic light leaks, and foreground elements responding to mouse movements and page scrolling.
- **Ambient Particles Canvas**: High-performance HTML5 Canvas animation generating drifting dust particles and soft glowing light leaks.
- **Video Reveal Page Load**: An elegant transition that fades the landing screen from darkness, creating a camera-reveal panorama effect.
- **Responsive Configurator**: Real-time package calculator adjusting participant count, accommodation tier, primary activities, and kosher dietary requirements.
- **Fluid Grid Layouts**: Fully responsive grid systems optimizing presentation layouts for both mobile viewports and high-resolution displays.
- **Premium Styling**: Sleek glassmorphism interfaces, neon accent glow filters, and customized scrollbars matching the luxury branding guidelines.

## Development

The project is built using **TanStack Start**, **React 19**, and **Tailwind CSS v4**.

To start the local development server:

```bash
bun run dev
```

To build the static production bundle:

```bash
bun run build
```
